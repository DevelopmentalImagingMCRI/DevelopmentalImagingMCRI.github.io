The Developmental Imaging Group at the Murdoch Childrens Research Institute works with MRI, CT and ultrasound imaging data for research studies. Software we release for processing such data is available from this site:

  • Melbourne Children's Regional Infant Brain (M-CRIB) atlas.
  • MANTiS: Morphologically adaptive neonate tissue segmentation. An SPM toolbox for tissue classification in T2 weighted MRI scans for neonates.
  • DWIMotionCorrect. A modified eddy/motion correction tool which reduces registration bias at b=3000.
  • freesurfer_statsurf_display. A set of tools for MATLAB that plots Freesurfer surfaces with each region of a selected cortical parcellation coloured according to p-value/effect size/arbitrary values/freesurfer colours/r squared.
  • Marker-based watershed scalper. A fast and robust data driven brain extraction tool for T1 weighted images based on the morphological watershed transform. Versions for human and macaque scans are included.
  • Supervised neighbourhood preserving embedding for detecting patterns of variation in MRI scans. Code supporting Ball et al.
